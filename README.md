## Description

[ReactJs](https://github.com/facebook/react) library starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# production build
$ npm run build

```

## Test

```bash
# unit tests
$ npm run test
```
